# element

## 安装创建vue项目和配置

一、安装
>npm install -g @vue/cli
>
>vue create vue-el-admin
>
>cd vue-el-admin
>
>npm run serve
二、配置vue.config.js(关闭保存代码错误提示)
module.exports = {
    lintOnSave:false
}
或vue ui（可视化管理项目）==》导入项目==》项目配置==》ESLint configuration==》关闭保存代码错误提示，会自动创建上述代码

## 安装element-ui和基础使用

vue add element
全局、覆盖元素的SCSS变量、中文
npm run serve
